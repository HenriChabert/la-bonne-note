See external ratings directly on food delivery, streaming, and hotel booking platforms.

La Bonne Note overlays ratings from popular review sources so you can check if a restaurant, movie, or hotel is worth it — without switching tabs. Ratings appear as small badges on every card, right where you need them.

FEATURES:
- Ratings from multiple providers displayed on every card
- Click any badge to open the original review page
- Filter by minimum rating or review count (hide or dim low-rated items)
- Enable or disable individual providers from the popup
- Change filters live — no page reload needed
- Results cached to save API quota

SETUP:
1. Click the extension icon and go to "Settings"
2. Enter your API keys for the providers you want to use
3. Browse a supported platform — ratings appear automatically

Full setup guide: https://github.com/HenriChabert/la-bonne-note/blob/main/docs/provider-setup.md

PRIVACY:
This extension does not collect any personal data. Item names are sent to rating providers solely to fetch ratings. API keys are stored locally in your browser.
